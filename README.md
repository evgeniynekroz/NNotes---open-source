## 🌟 ОБЗОР ПРОЕКТА

**NNotes** — это не просто очередной блокнот. Это сочетание высокой производительности и уникального визуального стиля. Приложение создано для тех, кто ценит эстетику **Dark Mode** и функциональность **Markdown**.

> **NNotes** (от **Nekroz Notes**) — умные заметки с поддержкой математики, рисования и многоуровневой защиты.

---

## 🚀 КЛЮЧЕВЫЕ ВОЗМОЖНОСТИ

### 🎨 Дизайн и Интерфейс
* **Liquid Glass UI:** Эффекты глубокого размытия (Blur), градиенты от черного к фиолетовому и минималистичные иконки.
* **Depth List:** Анимированный список заметок с динамическим изменением масштаба и прозрачности при скролле.
* **Smooth Transitions:** Плавные переходы между экранами через кастомные анимации.

### 🧠 Умный Функционал
* **Smart Math Solver:** Встроенный движок вычислений. Просто напишите уравнение (напр. 2+2=), и приложение предложит решение.
* **Markdown Engine:** Полная поддержка синтаксиса: заголовки, списки, ссылки, таблицы и блоки кода.
* **Version History:** Автоматическое сохранение до 5 предыдущих состояний каждой заметки для быстрого отката.
* **Canvas Drawing:** Режим свободного рисования с выбором цвета, толщины кисти и ластиком.

### 🛡 Безопасность и Организация
* **Biometric Auth:** Вход по отпечатку пальца или лицу (FaceID).
* **PIN Protection:** Возможность установки 4-значного кода доступа.
* **Folder System:** Удобная сортировка заметок по категориям.
